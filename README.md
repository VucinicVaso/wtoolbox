# WTOOLBOX


## ABOUT:
Mini framework za kreiranje mobilnih aplikacija. 
WToolBox sadrzi sledece servise:
- websocket servis
- oauth servis
- lokalna baza podataka
- document servis (upload, download fajlova, generisanje PDF-a)
- komponente za kreiranje UI-a
- theme servis
- translation servis
- validator servis
- MVC podrska
- ...
