# RouterRegisterService

**RouterRegisterService** sluzi za registrovanje ruta stranica aplikacija kojima glavna aplikacija moze da pristupi.
Rutiranje u aplikaciji se vrsi preko GetX library-a.
Servis sadrzi dvije klase:
- RouterRegisterService klasa koja sluzi za registrovanje BmPlRouting klasa,
- Routing klasa koja sluzi za registrovanje ruta aplikacije,
