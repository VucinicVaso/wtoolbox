# ComponentFactory

Servis ****ComponentFactory**** sluzi za kreiranje UI-a aplikacije koristeci ****component factory pattern****.

Sadrzaj servisa:
- ****factory****: sadrzi factory klasu kojom se kreira UI koristeci widgete iz component foldera,
- ****component****: sadrzi widgete koji sluzi za kreiranje UI-a,
- ****variant****: sadrzi enum klase sa tipovima komponenti
